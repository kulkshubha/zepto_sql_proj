📌 Zepto SQL Data Analysis
📖 Overview

This project focuses on analyzing e-commerce data from Zepto to extract meaningful business insights. The dataset was sourced from Kaggle and analyzed using SQL in pgAdmin.

🎯 Objectives
Understand customer purchasing behavior
Identify top-performing products
Analyze sales trends and patterns
🛠️ Tools & Technologies
SQL
PostgreSQL
pgAdmin
Kaggle
🔄 Project Workflow
Database & Table Creation
Data Import (CSV files)
Data Exploration
Data Cleaning (handling NULLs, duplicates)
Data Analysis & Business Insights
🔍 Key Insights
Identified top-selling products and categories
Analyzed customer purchase patterns
Discovered sales trends over time
⚠️ Challenges Faced
Handling missing and inconsistent data
Writing efficient queries for complex analysis
Managing duplicates and data quality issues
📊 Conclusion

This project demonstrates how SQL can be used to clean, analyze, and extract insights from real-world datasets, supporting data-driven decision-making.
